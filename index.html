Changes to Font

To change font style, use the following:

font-family:Sans-serif

Here, Sans-serif is just one example font style you can change to.

    Others include:
  • Helvetica
  • Arial
  • Verdana
  • Tahoma
  • Gill Sans

Background Color

To change the background color of the element use the following:

background-color:#FFFFFF

You can use hex codes, rgb styling, or simple color words to choose the color for your background.

Margins

To change the side of the margins of your element, use one of the following:

Margin: 0 0 0 0; Margin: 0 0 0; Margin: 0 0; Margin: 0

When using four seperate values, you are choosing the margin size for top, right, bottom, left in that order.

With three values, you are choosing top, left and right together, and finally bottom.

With two values, you are choosing top and bottom together, followed by left and right together.

With one value, you are applying it to all margins of the element.

Using Class Selectors in CSS

When you want to change all elements of a certain class type, use the following:

.Class { }

Layering Elements

To manipulate which elements are on top or under other elements, use the following:

z-index:1

By giving an element a z-index value higher than another element, you will ensure that the higher valued one will be on top of the other element.
